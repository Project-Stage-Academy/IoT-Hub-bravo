# Communication Channels and Escalation Paths

## Team Communication Channels
| Purpose                       | Channel / Tool             | Notes |
|--------------------------------|---------------------------|-------|
| General team discussion         | Discord #team-bravo          | Daily updates, quick questions |
| Technical questions / support   | Discord #team-bravo     | Ask for help on code, setup, or errors |
| Project announcements           | Discord #team-bravo    | Important updates from Product Owner or Leads |
| Code review / PR discussions    | GitHub Pull Requests / Discord #bravo-github-gifts      | Use comments to provide feedback |
| Documentation updates           | GitHub Issues / PRs / Discord #team-bravo | Notify when docs are added/updated |
| Emergency / urgent blockers     | Discord direct message /  #team-bravo    | Notify everyone or mentor |

## Escalation Paths
1. **Blocker / Critical Issue**
   - Contact: Relevant intern
   - If unresponsive within 2 hours: escalate to Mentor
2. **General Question**
   - First: ask in the appropriate Discord channel
   - If unresolved within 24 hours: escalate to Mentor
3. **Documentation or process question**
   - First: check docs
   - Then: ask Documentation Owner
   - Escalate to Product Owner if clarity not achieved

## Notes
- Always mention the relevant tag/channel when posting.
- Keep communication professional and concise.
- Document important decisions in GitHub Issues or project documentation.
